넌센스 퀴즈 모음 사이트를 만들고 싶어.
- React Native를 이용해서 만들어 줘
- 메뉴는 세 가지가 있어. 퀴즈 모음 검색, 퀴즈 맞추기, 퀴즈 등록이야.
- DB는 Supabase를 이용할게

1. 퀴즈 모음 검색
  : 검색창이 있고, 텍스트를 입력할 수 있어요. 입력한 텍스트를 기준으로 넌센스 퀴즈 질문에서 검색해서 결과를 보여줘야 해.

2. 퀴즈 맞추기를 시작하면, 질문이 나오고 답을 맞춰야 해
 : 답을 맞추는 방식은 입력한 답이 띄워쓰기 단위로 나눠져서, 정답이 아닌 단어와 같이 섞여서 바둑판처럼 노출이 되야 해. 순서대로 정답을 선택하면 맞추는 거야.
 : 퀴즈를 맞추거나 틀리면, 이 문제가 괜찮은지 '좋아요'와 '싫어요', '패스' 중에 하나를 선택할 수 있어. 이 결과는 DB에 저장돼야 해.
 : 좋아요 스코어가 높은 퀴즈는 다음에 노출될 확률이 높아야 해.
 : 다 끝나면 이름을 입력 받고, 스코어를 알려줘야 해. 스코어보드가 있어.

3. 퀴즈 등록
 : 등록 버튼이 있어서, 넌센스 퀴즈 문제(질문과 답)를 직접 등록할 수 있어야 해
 : 등록한 문제는 관리자가 승인한 후에 사람들에게 보여
